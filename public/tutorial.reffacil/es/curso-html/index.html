Introducción

HTML es uno de los primeros lenguajes que se deben conocer para introducirse en el maravilloso mundo del desarrollo web. Puede decirse hoy, con toda seguridad, que la web es una tecnología que ha permitido que la información llegue a casi todas las personas del mundo.

Objetivos

  • Crear a través del tiempo una base de información sobre HTML que albergue sus principales cracterísticas y casos de uso.
  • Conocer y dominar HTML. Aprender en que momento utilizar una u otra, las características de HTML.

¿Para quienes es HTML?

El lenguaje HTML es fácil de aprender y puede ser dominado por una amplia gama de profesionales, puesto que su objetivo principal es el de presentar la información de la forma más estructurada y ordenada posible. Es debido a esto que el contenido sobre HTML presentado aquí pretende alcanzar a:

  • Profesionales del marketing digital, que deseen mejorar o editar la presentación de sus campañas digitales.
  • Creadores de contenido SEO, por sus siglas en inglés, Search Engine Optimization (Optimización en motores de busqueda) que desean presentar la información cumpliendo con los estándares o mejores prácticas de presentación de contenido para motores de busqueda y asi tener más relevancia en los resultados.
  • Profesionales de la educación que deseen crear un curso HTML o material de estudio sobre HTML para reforzar conocimientos.
  • Personas que deseen convertirse en desarrolladores de paginas web o aplicaciones web, es decir que desean volcar su creatividad escribiendo su propio código.
  • Profesionales en general que desean ampliar su alcanze al presentar sus habilidades a través de la web.

¿Para qué sirve HTML?

El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar una página web y sirve para desplegar la página web y sus contenidos. HTML es el lenguaje que se utiliza para el desarrollo de páginas de internet. Está compuesto por una seríe de bloques de código conocidos como etiquetas que el navegador interpreta y da formato en pantalla. HTML dispone de diferentes etiquetas para los elementos que se desean introducir a las páginas web.

Breve historia de la web

Antes de empezar en el desarrollo web ayuda conocer algunas diferencias básicas sobre lo que es internet y sobre lo que es la web. Internet y la web son dos tenologías difrentes entre si a pesar de trabajar juntas.

  • Internet es un conjunto desentralizado de redes de comunicación de diferentes tipos que pueden ser o estar interconectadas entre si, internet permite que estas redes físicas heterogéneas, a través de diferentes protocolos de comunicación, puedan intercambiar datos. Es correcto cuando se piensa en internet imaginar su infraestructura (Antenas, routers, cables submarinos, computadores, dispositivos móviles, satélites, etc...). Un protocolo de comunicación que más se ha popularizado entre los usuarios de internet es el http o https, uno de los más utilizados por la web.
  • Mientras que la web forma parte de internet, siendo una forma de consulta remota de archivos de hipertexto.

El primer servidor web se activo el 20 de diciembe de 1990. Tim Berners Lee, es considerado el padre de la web ya que desarrollo:

  • HTML, siglas en inglés de HyperText Markup Language (Lenguaje de marcas de hipertexto).
  • HTTP, siglas en inglés de HyperText Transfer Protocol (Protocolo de transferencia de hipertexto).
  • URL, siglas en inglés de Uniform Resource Locator (Localizador de recursos uniforme).

¿Cuáles son los requerimientos para crear una pagina web?

Para crear una pagina web solo se necesita

  • Un editor de texto: Para escribir el código HTML. Y guardar lo escrito bajo la extensión .html
  • Un navegador web: Como podria ser chrome, firefox o safari para interpretar el código HTML. Deberá abrir la pagina con el navegador web.

Pero si desea que el acceso a la página web sea público deberá implementar unos pocos requerimientos más.

  • index.html: Para abrir la página en un computador local no es necesario crear la página index.html, pero si se desea desplegar en un servidor web entonces si es necesaria. A través del servidor se puede configurar que la página de inicio no sea index.html pero no es lo habitual.
  • Un servidor web: Donde podrá alojar su sitio. y que el acceso sea público.
  • Un nombre de dominio: que posee el formato https://wwww.mi_nombre_de_dominio.com y lo diferencia del resto de otras páginas web creadas de acceso público.

¿Qué es una página web?

Las paginas web son documentos a los que se puede acceder a través de un enlace que en la mayoría de los casos posee el formato: https://www.nombre_de_dominio.com; donde https hace referencia al protocolo de acceso aunque no se limita unicamente a este, nombre_de_dominio es el nombre que escojió el usuario como nombre de su presencia en la web y la extensión .com viene a ser el nivel(alcanze) del dominio, siendo este el más común de entre muchas extensiones. Algo esencial de la web son los enlaces o hipervínculos.

¿Qué son los enlaces o hipervínculos?

Los enlaces o hipervículos se definen mediante una URL. Los enlaces en una pagina web pueden ser externos o internos. Los enlaces internos por lo general poseen el formato file:///Usuario/Documentos/mipagina.html. Mientras que los enlaces externos poseen el formato que incluye al inicio un protocolo de acceso a internet como lo puede ser, por ejemplo, http, https o ws.

Qué es HTML

HTML son las siglas de (Hyper Text Mark Language) ó lenguaje de marcado de hipertexto, es una de las principales tecnologías para la construcción de un sitio o una aplicación web. Con HTML es posible definir la estructura de una página web.

HTML no es un lenguaje de programación; es un lenguaje de marcado que define la estructura de tu contenido. HTML consiste en una serie de elementos que usarás para encerrar diferentes partes del contenido para que se vean o comporten de una determinada manera. Las etiquetas de encierre pueden hacer de una palabra o una imagen un hipervínculo a otro sitio, se pueden cambiar palabras a cursiva, agrandar o achicar la letra, etc.

¿Qué son entidades de carácter?

Las entidades de carácter son referencias a un carácter y se representa con la sintaxis &nombre;. Para mostrar el código html como texto utilizaremos éstas dos entidades de carácter:

Una entidad HTML es un conjunto de caracteres ("string") que comienza con un ampersand ( & ) y termina con un punto y coma (;) . Las entidades son utilizadas frecuentemente para imprimir en pantalla caracteres reservados (aquellos que serían interpretados como HTML por el navegador) o invisibles (cómo tabulaciones).

Etiquetas HTML

Las etiquetas HTML son pequeños bloques de código, que indican al navegador como debe interpretar el contenido recogido entre dichas etiquetas. Por ejemplo, si queremos «pintar» un párrafo de texto, hay una etiqueta especifica para que el navegador interprete ese texto como un párrafo.

Estas etiquetas cuentan además con atributos que podemos añadir y que en sí, pueden determinar como será el comportamiento específico que tendrá la etiqueta. Te dejo por aquí un listado de atributos HTML para complementar esta entrada.

Las etiquetas HTML comienzan siempre con el símbolo «<» y finalizan con el símbolo «>». Entre medias de estos dos símbolos irá el nombre de la etiqueta que queremos que el navegador interprete.Todas las etiquetas HTML están compuestas por una etiqueta de apertura, y una etiqueta de cierre. Aunque hay excepciones, como por ejemplo la etiqueta para cargar una imagen que son llamadas etiquetas huérfanas, ya que solo tienen etiqueta de apertura. Las etiquetas de cierre se construyen con una barra «/» justo antes de la propia etiqueta y los símbolos «<» y «>». Resumiendo, si queremos «pintar» un párrafo en la página web, abriremos con la etiqueta de apertura de párrafo <p>», y cerraríamos con la etiqueta de cierre «</p>»

Tipos de etiquetas HTML

Elementos de Línea (Inline) y Elementos de Bloque (Block)

A medida que se va estructurando un sitio web introduciendo etiquetas HTML al código, los navegadores de forma predeterminada otorgarán ciertos estilos o comportamientos a estos elemetos HTML sin que el programador los haya definido previamente. Estos estilos predeterminados forman parte de la interpretación que hace un navegador web de una etiqueta HTML en particular.

Uno de los pricipales comportamientos que otorga un navegador web a una etiqueta HTML es determinar si dicha etiqueta HTML es un elemento de bloque o es un elemento de línea. Aunque este comportamiento es uno de los pricipales que determina un navegador web es pasado por alto en algunos casos, no logrando encontrarse la solución a cierto problemas de maquetación o implementando código CSS inecesario o complejo para resolver el problema.

Aunque con CSS es posible definir su una etiqueta HTML será un elemento de línea o un elemento de bloque, para que puedas ir creando una estructura HTML adecuada en tus proyectos o mejorarla, ten siempre en cuenta que los navegadores dependiendo de la etiqueta HTML que uses van a definir de manera predeterminada si es un elemento de línea o de bloque. Conociendo esto quizás puedas ahorrarte algunas líneas de código en tus estilos CSS.

Elementos de Línea

Una etiqueta o elemento de línea es aquel que ocupa el espacio mínimo necesario en horizontal, y permite que otro elemento se coloque a su lado. En general, los elementos en línea pueden contener únicamente los datos y otros elementos en línea; un elemento en línea ocupa sólo el espacio delimitado por las etiquetas que envuelven su contenido y de forma predeterminada, no comienzan con la nueva línea. A continuación encontrarás todas las etiquetas HTML consideradas elementos de línea por los navegadores web.

Elementos de Bloque

Una etiqueta o elemento de bloque, ocupa todo el ancho disponible y no permite que otro elemento se coloque a su lado (aunque aparentemente tenga lugar suficiente). Un elemento en bloque ocupa todo el espacio de su elemento padre (contenedor), creando así un "bloque". En general, los elementos en bloque pueden contener elementos en línea y otros elementos en bloque y de forma predeterminada comienzan en una nueva línea. A continuación encontrarás todas las etiquetas HTML consideradas elementos de bloque por los navegadores web.

Un elemento de bloque puede contener uno o varios elementos de bloque o elementos de línea, pero un elemento de línea debería solo contener, de ser necesario, otros elementos de línea.

Atributos HTML

Los atributos en HTML son elemento muy utilizado dentro de las etiquetas HTML, un atributo HTML añade una clave a la etiqueta HTML que puede tener o no un valor especificado.

La forma de añadir un atributo html a una etiqueta sería <etiqueta_html atributo = "valor"> Contenido </etiqueta_html>
